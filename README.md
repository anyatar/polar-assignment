# Rest API with MySQL 

| Methods	| Urls	          | Actions
| --------- | ----------------| ----------------------------------------- |
| POST      | /signup         | Signs the user up to the application
| PUT       | /update         | Updates the users total running distance
| GET       | /mystats        | Returns the users' ranking
